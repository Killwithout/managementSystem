"# managementSystem" 
这是一套后台管理系统的java小项目，里面的功能大多没有实现，前端js部分也不太完整
前台技术：html + jquery + ajax + css 
后端框架：Spring + springMVC + Mybatis
目前实现了登录功能和登陆后的页面的一些跳转，在登录的功能中添加了一个图片验证码，登录密码利用MD5加密
